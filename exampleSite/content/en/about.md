### About the Theme

The Primer Blog theme is a clean, responsive Hugo theme that uses GitHub's Primer CSS design system. It features:

- Dark mode support with system preference detection
- Full multilingual support
- Responsive sidebar layout
- Tag and archive navigation
- Social sharing buttons
- Pagination
- Related posts
