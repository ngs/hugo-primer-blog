## このブログについて

これは Hugo と Primer Blog テーマで構築されたサンプルブログです。

### テーマについて

Primer Blog テーマは、GitHub の Primer CSS デザインシステムを使用したクリーンでレスポンシブな Hugo テーマです。以下の機能があります：

- システム設定検出付きダークモードサポート
- 完全な多言語サポート
- レスポンシブサイドバーレイアウト
- タグとアーカイブナビゲーション
- ソーシャル共有ボタン
- ページネーション
- 関連記事
